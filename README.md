# vue-framework

基于vue3 + vite + element-plus搭建的前端通用项目后台框架

## 初始化项目

```
yarn
yarn dev

// 打包
yarn build
```

### 基于vscode的插件依赖
> 保证统一格式化风格和自动校验格式

ESLint、Prettier ESlint、Prettier、Vue Language Features
